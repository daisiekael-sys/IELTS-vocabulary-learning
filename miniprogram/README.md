# 雅思词汇学习小程序

基于微信小程序开发的雅思词汇学习应用，包含高频同义替换和艾宾浩斯记忆训练两大核心功能。

## 功能特点

### 1. 高频同义替换
- 10个单元，80+词汇分组
- 正常/隐藏中文两种学习模式
- 单星/双星标词功能
- 按星标过滤词汇
- 本地数据持久化

### 2. 艾宾浩斯记忆训练
- 基于遗忘曲线的智能复习系统
- 4大分类：科技、文化、工作、生活
- 7个复习间隔：20分钟、1小时、9小时、1天、2天、6天、31天
- 学习进度统计
- 待复习单词提醒

### 3. 数据管理
- 学习数据本地存储
- 数据导出/导入功能
- 一键清除数据

## 技术栈

- 微信小程序原生开发
- WXML + WXSS + JavaScript
- 本地存储 (wx.getStorageSync)

## 项目结构

```
miniprogram/
├── app.js              # 小程序入口
├── app.json            # 全局配置
├── app.wxss            # 全局样式
├── sitemap.json        # 站点地图
├── pages/
│   ├── index/          # 首页
│   ├── synonyms/       # 同义替换页面
│   └── ebbinghaus/     # 艾宾浩斯页面
├── utils/
│   ├── vocabularyData.js   # 词汇数据
│   └── storage.js          # 存储工具
└── images/             # 图标资源
```

## 使用方法

1. 使用微信开发者工具打开 `miniprogram` 目录
2. 填写自己的 AppID（或选择测试号）
3. 编译预览

## 数据存储说明

所有学习数据存储在本地，使用 key: `ielts_learning_system`

数据结构：
```json
{

    "masterMap": {},
    "stats": {},

    "memory": {}
  },

    "starred": {}
  },
  "settings": {}

```
